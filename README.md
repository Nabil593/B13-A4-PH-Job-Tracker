# Questions & Answer

## 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
### Answer: getElementById gets the id from the element. The id must be unique name. getElementsByClassName gets multiple elements by class name. querySelector gets first element from CSS selector and  querySelectorAll gets all elemnts from css selector.




## 2. How do you create and insert a new element into the DOM?
### Answer: I create an elements useing createElement() and insert it into the DOM useing appendChild() or append().




### 3. What is Event Bubbling? And how does it work?
Answer: Event Bubbling is an event which is work start from child element and moves up to parent element.
It works by triggering the event first child element and then moves up to the parent element one by one.




### 4. What is Event Delegation in JavaScript? Why is it useful?
Answer: Event delegation is when we put one event on the parent rather than adding events to all child elements. This is a JavaScript technique.
It is useful because it saves memory, it works for new elements and it makes code easier.




### 5. What is the difference between preventDefault() and stopPropagation() methods?
Answer: preventDefault() is stops the browser's default action. On the other hand, stopPropagation() is stops the event from bubbling up the DOM.
